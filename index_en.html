With the latter, in addition to programming knowledge, I have acquired knowledge in different tools such as Node or Eslint, the connection with databases, relational or not, etcetera.

Projects

Recommender system

A recommender system based on collaborative filtering, in which, given a matrix with the ratings of multiple user i on item j, it seeks to fill the gaps in the matrix, thus offering a recommendation of items to each user.
With this, the aim is to be able to make an approximation of the valuation that the user would make, as accurate as possible to reality, in order to be able to select a posteriori, the items that the user will be more interested in, and thus achieve greater user satisfaction.
The development is done in typescript and has a web user interface.

100%

Sudoku Solver

Artificial Intelligence project using Prolog and Python to develop a desktop application capable of solving sudokus given by the user. The program has a user interface that offers a board and different game modes to solve. The user will enter the values of the board to be solved and the program will solve it. The interface is implemented in Python, while the board logic has been implemented using the declarative language Prolog.

Cipher Tools

Project in which many of the most common ciphers, both past and present, are put together. The development has been carried out mostly in Python, and in some cases, a graphical interface is available.
The project is not finished, but is stopped in a development phase.

A Star Pathfinder

A* algorithm implementation for the problem of finding the solution of a labyrinth. The algorithm is based on different euristics, such as the Manhattan distance, the Euclidean distance or the Chebyshev distance.
The development is done entirely in C++ as a collaborative work with my colleagues Dana Belén Choque and Stephanie Andreina.

Maze Solver

Implementation of a recursive algorithm for solving mazes in C++. The algorithm is based on the depth-first search algorithm applied to a maze.
The development is done entirely in C++.
